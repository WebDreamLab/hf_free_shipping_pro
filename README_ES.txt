FREE SHIPPING PRO MODULE BY HFMODULES

Este módulo permite:
  - Definir el porte gratis para un producto, o bien definir un coste de envío fijo por producto, zona y transportista.
  - Definir los portes gratis para los productos de una categoría según la zona, portes fijos para todos los productos de 
    una categoría y para cada zona y transportista; definir un importe para cada categoría y zona, superado el cuál, 
    los portes son gratis para los productos en el pedido de esa categoría.
  - Idem para los proveedores y fabricantes.

En el resumen del carrito se muestran los importes que falta de cada categoría, proveedor y fabricante para que los portes
sean gratis.
Si el cliente está logueado, en la ficha del producto aparece un texto indicando las condiciones para que los portes de
ese producto sean gratuitos.
En la configuración del módulo puede seleccionar que transportistas realizarán el envío en el caso de que haya o no
productos con envío gratuítos en el pedido.

Basta con instalar el módulo, y las opciones de gastos de envío gratuítos o fijos 
apareceran en la ficha del producto en el panel del backoffice, en la ficha de las categorías y en la de 
proveedores y frabricantes.
Si selecciona un precio fijo, debe establecer el precio para cada zona y transportista.
Si deja a celda transportista/zona vacía, el módulo no actuará para esta combinación transportista/zona

Esta plantilla de módulo es configurable via un css específico, y también mediante los tpl's para la 
plantilla de producto FrontOffice, carrito de compra, y la plantilla de la lista de productos.
Puede modificar estos archivos para adaptarlos al template de su comercio.

REQUISITOS

Tener permisos de escritura en la carpeta "override/classes".
Este módulo no es compatible con la versión normal de este módulo "hf_free_shipping".
